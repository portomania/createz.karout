# CREATEZ

Geschäftsführer-Cockpit der CREATEZ GmbH – mobile Portal-App (PWA) mit
Einsatz-Kalender, Aufgaben, Finanzen und Notizen. Offline nutzbar und
installierbar auf dem Smartphone.

- `app.html` – die App (Login-PIN, Kalender, Aufgaben, Finanzen, Notizen)
- `index.html` – Startseite mit Button „Cockpit öffnen"
